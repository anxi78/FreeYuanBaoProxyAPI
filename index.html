FreeYuanBaoProxyAPI

OpenAI 兼容的腾讯元宝 HTTP API 代理服务，提供聊天、图片生成等功能，无缝对接现有 OpenAI 生态工具。

v1.0 · OpenAI Compatible

核心特性

OpenAI 兼容

完全兼容 OpenAI Chat Completions API，可使用任何支持 OpenAI 标准的客户端直接接入。

图片生成

支持 /v1/images/generations 标准接口，通过元宝 Bot 生成图片后返回 COS 预签名直链。

工具调用

支持 tool_calls / function calling，可将外部工具能力注入到元宝 Bot 的对话流程中。

Web 管理面板

内置 Web 管理界面，支持在线查看和修改配置、聊天测试以及服务重启。

API 端点

GET /v1/models
获取可用模型列表
POST /v1/chat/completions
聊天补全，支持 streaming、temperature、top_p 等标准参数，支持 tool_calls
POST /v1/images/generations
图片生成，支持 size 参数（如 1024x1024），返回图片直链

技术栈

Python 3 asyncio WebSocket Protobuf OpenAI API Schema
